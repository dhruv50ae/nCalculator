## Description

- Simple web app for calculating the addition of two numbers
- Uses Node.js
- Uses Express.js
- Uses Body-parser
- Uses post and get to as routes

## What I learnt

- Node.js ✅
- Express.js ✅
- Body-parser.js ✅

## How To Use

- Execute this command to start: node calculator.js
- Enter two numbers and hit enter
